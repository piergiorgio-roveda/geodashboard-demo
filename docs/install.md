# Install

## Requirement

* Apache
* Php
* Postgres, PostGIS, pgRouting
* Mysql
* Geoserver

-----
In order to use this app you can either:

* install Wordpress
* overwrite wp-content folder with this repository /trunk/wp-content
* rename original wp-config.php file in wp-config-original.php
* copy in root the files in /trunk/
  * env-h3.php
  * wp-config-extends.php
  * wp-config.php
