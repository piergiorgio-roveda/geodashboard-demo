== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "addon-geo"
3. **Activate** addon-geo from your Plugins page
4. **Click** load a page for plugin result!
5. **Read** the documentation to [get started](https://.../)
